# Lost and Found App

## Overview
This is an Android mobile application developed using Java and SQLite. The app allows users to report lost or found items and helps connect items with their owners.

## Features
- Create lost/found adverts
- Upload images for items
- Store data using SQLite database
- View all items in a list
- Search/filter items by category
- View item details
- Delete items after recovery

## Technologies Used
- Java
- Android Studio
- SQLite Database

## How to Run
1. Open project in Android Studio
2. Run on emulator or Android device
3. Test all features
